# Biblioteca de Videojuegos

Aplicación web para gestionar una biblioteca personal de videojuegos, agregar reseñas y visualizar estadísticas.

## Tecnologías utilizadas

### Backend
- Node.js
- Express
- MongoDB (con Mongoose)

### Frontend
- React
- CSS

## Funcionalidades

- Gestión de videojuegos (agregar, editar, eliminar)
- Sistema de reseñas
- Filtrado y búsqueda
- Dashboard de estadísticas personales

## Instalación

1. Clonar el repositorio
2. Instalar dependencias del backend: `cd backend && npm install`
3. Instalar dependencias del frontend: `cd frontend && npm install`
4. Configurar variables de entorno en un archivo `.env`
5. Iniciar el servidor: `cd backend && npm start`
6. Iniciar el frontend: `cd frontend && npm start`

## Estructura del proyecto

- `/backend`: API REST con Express y MongoDB
- `/frontend`: Aplicación React
